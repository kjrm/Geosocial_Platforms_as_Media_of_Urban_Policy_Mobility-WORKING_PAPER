# Geosocial Platforms as Media of Urban Policy Mobility

This repository includes the code and data or data source used to complete the three studies included in the working paper 'Geosocial Platforms as Media of Urban Policy Mobility'.

It is organized into three folder, one per study.
- `study_1`: Platform as feedback tool
- `study_2`: Platform as policy implement
- `study_3`: Platform as facilitator of sentiment and idea flow

Each folder includes a `README.md` to orient access to the files.

Thank you to Luke Pan (https://github.com/Xera-phix/platform-as-urban-policy-mobility-medium/tree/master) for help protoyping parts of the studies.
