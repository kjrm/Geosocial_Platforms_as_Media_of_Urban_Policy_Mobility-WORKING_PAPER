# sentiment flows

This repository contains a python notebook (`study3-1.pynb`) with the code, data (`PA_brewpub_reviews.csv`), and shapefiles (`PA Municipality Boundaries_20260120.zip`) necessary to complete the first part of the study. This part tracers and tallies users' inter-municipal review sequences, using Google reviews of Pennsylvania brewpubs.

The notebook relies on data made accessible by Mcauley lab (https://mcauleylab.ucsd.edu/public_datasets/gdrive/googlelocal/). Only the Pennsylvania datasets are used in this study. 
The data included in this repository (`PA_brewpub_reviews.csv`) is a subset of the data aquired from this source, only including reviews of Pennsylvania businesses which include "brewpub" in the category field.

The study also relies on Pensylvania municipality shapefiles (`PA Municipality Boundaries_20260120.zip`) made availble through the PA Open Data Portal (https://data.pa.gov/Geospatial-Data/PA-Municipality-Boundaries/ednf-5bi6/about_data).
