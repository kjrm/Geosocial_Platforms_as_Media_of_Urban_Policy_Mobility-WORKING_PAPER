# sentiment flows

This repository contains a python notebook (study3-1.pynb) with the code necessary to complete the first part of the study, which tracers users' inter-municipal review sequences, using Google reviews of Pennsylvania brewpubs.

The notebook relies on data made accessible by Mcauley lab (https://mcauleylab.ucsd.edu/public_datasets/gdrive/googlelocal/). Only the Pennsylvania datasets are used in this study.
