# Study 3: Platform as facilitator of sentiment and idea flow

This repository contains three folders for completing three different parts of the study:
- `1-sentiment_flows` : using Google reviews of Pennsylvania brewpubs, and tracing users' inter-municipal review sequences.  
- `2-suggestion_labelling` : using an ensemble of LLM models to label a subset of reviews according to whether they offer explicit suggestions for improvement.   
- `3-suggestion_map` : map the inter-municipal review sequences of users who have at least one review which includes an explicit suggestion for improvement
